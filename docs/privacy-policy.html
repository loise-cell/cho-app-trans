隱私權政策

最後更新日期：2026-07-11

Cho App Trans（以下稱「本 App」）由開發者營運。使用本 App 即表示您已閱讀並理解本政策。

1. 我們蒐集哪些資料

類型 說明 是否上傳至伺服器
相機／相簿圖片 您主動拍攝或選取，用於 OCR 與翻譯 是（經由我們的 API 代理轉送至 OCR 服務）
辨識與翻譯文字 OCR 原文與翻譯結果 是（翻譯流程中）
點數與徽章 儲存在裝置本機 否（目前版本）
語言偏好 介面與翻譯語言設定 否（儲存在裝置本機）
廣告互動 Google AdMob 獎勵廣告觀看事件 由 Google 依其政策處理

我們不要求您註冊帳號，也不主動蒐集姓名、電話或精確定位。

2. 資料用途

  • 提供圖片文字辨識（OCR）、翻譯與單字查詢
  • 透過觀看獎勵廣告發放點數，並在翻譯時扣點
  • 防止濫用（API 代理限流）

3. 後端代理與第三方服務

為保護 API 金鑰與實施限流，OCR 與翻譯請求會先經由 Cloudflare Workers（我們的 API 代理）轉送，再交由下列第三方處理：

服務 用途
OCR.space 圖片文字辨識
MyMemory (Translated.net) 機器翻譯
dictionaryapi.dev 英文單字查詢（於 App 內查詢時）
Google AdMob 獎勵式廣告

我們不會將您的內容用於訓練模型或再行銷。代理伺服器不長期保存您的圖片或翻譯內容。

4. 資料保存與刪除

  • 翻譯紀錄僅暫存於 App 記憶體，關閉後可能清除
  • 點數與設定保存在裝置本機，解除安裝即刪除
  • 刪除相關資料請來信聯絡我們

5. 兒童隱私

本 App 並非專為 13 歲以下兒童設計。
